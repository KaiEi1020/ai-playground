## 1. 结构 (Structural)

### 类型
实操手册 / 投资者教育 / 财报排雷指南

### 一句话主旨
以投资者视角,手把手教读者通过阅读上市公司财报排除造假企业和竞争力低下的企业,而非挖掘牛股。

### 骨架 (主要论点及其关系)

1. **财报是用来排除企业的** — 全书核心指导思想。财报的功能是提示风险、排除造假,而非挖掘牛股。
2. **资产负债表是高手最关心的表** — 资产按生产/经营/投资三类分类解读,逐项揭示造假信号。
3. **利润表是资本市场追捧但最容易造假的表** — 权责发生制是造假温床,需警惕收入确认、毛利率异常等。
4. **现金流量表是企业现金流肖像** — 经营/投资/筹资三类活动的现金流组合揭示企业真实状况。
5. **财务指标分析和估值** — 用 ROE、杜邦分析等指标辅助判断,但估值只是艺术。
6. **财报附注和董事会报告是藏雷区** — 企业必须披露却不想让你明白的东西藏在附注里。
7. **常见操纵财报手法与反查** — 系统总结造假手段和识别痕迹。

**论点之间的关系**: 层层递进 — 从理念(排除企业)→工具(三张表)→实操(逐项排雷)→综合(指标与估值)→反诈(造假识别)

### 作者要解决的核心问题
普通投资者如何在不具备会计专业背景的情况下,读懂上市公司财报,识别财务造假信号,排除不值得投资的企业。

## 2. 解释 (Interpretive)

### 关键术语 (作者本人的用法)

| 术语 | 作者的定义 | 和常识用法的差异 |
|---|---|---|
| 财报是用来排除企业的 | 财报的核心功能是提示风险、排除造假企业,而非挖掘牛股 | 常识认为财报是用来发现好公司的 |
| 中翻中 | 将艰涩深奥的会计准则表述转化为简单易懂的日常语言 | 作者自创术语,强调通俗化翻译 |
| 货币资金充裕倍数 | 货币资金余额 / (投资支出+分配支出),衡量资金运用效率 | 作者自创指标 |
| 存贷双高 | 货币资金充裕却借入大量有息负债,是资金虚构/挪用的信号 | 财务分析术语,但作者强调其排雷价值 |
| 归母净利润 | 合并利润表中减去少数股东损益后,属于上市公司股东的净利润 | 投资圈通用简称,作者详细解释其计算 |
| 经济商誉 | 企业长期竞争中建立的无法单独辨认但能为企业带来超额利润的能力 | 借用巴菲特概念,与会计商誉区分 |
| 权责发生制 | 以权利或责任的发生来确定收入和费用,不考虑实际现金收付 | 强调这是财务造假的"温床" |
| 摊余成本 | 将持有期总收益按实际利率分摊到各期的计量方法 | 用通俗例子解释专业概念 |

### 核心命题 (用自己的话)

1. 投资不能只看利润数据,要了解利润的产生过程,否则容易将财务把戏视为利润增长。
2. 经验丰富的投资者首先看资产负债表,而非利润表 — 资产负债表反映企业"家底"。
3. 只看公司利润数据很容易被误导;财报中有很多细节可以帮助我们排除造假企业。
4. 公司改变会计政策是大事儿,遇见一定要打起精神思考原因和影响。
5. 事有反常必为妖 — 财报中的反常现象是远离信号,投资应遵循"疑罪从有"原则。
6. 优秀上市公司通常有个特点:"其他xx"科目涉及金额较小,甚至为零。
7. 如果厨房里发现一只蟑螂,你很快就会遇到它的亲戚们 — 造假行为很少是孤立的。
8. 股市垃圾十之八九,多看一二 — 没有必要将财富下注在自己不信任的人身上。

### 论证链

作者以贵州茅台 2013 年年报为范本,逐科目讲解含义和阅读要点。每个科目都遵循"是什么→怎么看→造假信号→案例"的结构。论证路径: 先建立"排除企业"的理念框架 → 以三张表为结构逐项拆解 → 每个科目给出"正常情况"和"危险信号"的对比 → 用真实案例(如蓝田造假、银广夏造假等)佐证 → 最后汇总成"快速阅读"和"反诈"的实操指南。

## 3. 批判 (Critical) ★

### 作者的时代局限
- 本书以 2014 年贵州茅台财报为主要范本,虽然补充了新准则内容,但核心案例仍停留在 2013-2014 年。中国股市和会计准则此后又有变化(如注册制全面推行、退市新规等)。
- 作者的投资框架主要适用于 A 股市场,对港股、美股等境外市场的适用性有限(作者也明确提醒了这一点)。
- 书中对"新准则"的阐述截至 2020 年,2021 年后会计准则又有更新。

### 作者的立场盲点
- 作者自称"非会计专业人员",所有知识来自自学,这既是优势(通俗易懂)也是局限 — 某些会计处理的解释可能不够严谨。
- 作者的投资风格偏向价值投资(长期持有、满仓、不用杠杆),对短线交易、量化投资等策略缺乏覆盖。
- 作者以贵州茅台为"完美范本",但茅台是极为特殊的商业模式(产品供不应求、先款后货、存货不贬值),其财报特征不能简单套用到其他行业。
- 作者对"排除企业"的理念强调有余,但对"如何确认好企业"的系统性框架不足 — 读完本书知道什么不能买,但买什么仍需要其他知识补充。

### 未被证明的假设
- 假设"标准无保留意见"的审计报告才有阅读价值 — 但现实中也有"标准无保留"后暴雷的案例。
- 假设管理层诚信度可以通过财报细节推断 — 但高级造假者可以精心布置,让财报看起来"正常"。
- 假设投资者有足够时间和能力逐项阅读财报 — 对于覆盖大量标的的机构投资者或时间有限的个人投资者,这种"逐项排雷"的方法成本过高。

### 最强反对意见
"财报分析只是后视镜,等你能从财报中看出问题时,股价已经跌了一半。真正创造价值的投资,是前瞻性地判断行业趋势、技术变革和管理层能力,这些都无法从财报中直接读出。过度依赖财报排除法,可能让你错过高成长企业(它们早期财报往往并不好看),也可能让你陷入'过度分析'的陷阱。"

> **以上批判会直接成为下游 skill 的 Boundary (B) 字段来源**
