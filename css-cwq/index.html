CSS cqw 测试

CSS 里有效的是 cqw，不是 cwq。1cqw 等于查询容器宽度的 1%。拖动右侧面板宽度，标题、文字、盒子尺寸会跟着容器变化。

Container Query Units

先给父元素设置 container-type: inline-size，子元素就可以使用基于容器尺寸的单位。

  • cqw容器宽度 1%
  • cqh容器高度 1%
  • cqiinline 方向 1%
  • cqbblock 方向 1%
  • cqmin较小边 1%
  • cqmax较大边 1%

拖动这里改变容器宽度

下面的内容使用 font-size: clamp(24px, 10cqw, 88px)。
