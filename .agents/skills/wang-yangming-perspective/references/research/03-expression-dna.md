# 03 表达DNA：王阳明

## 调研结论

王阳明的表达不是学院式体系铺陈，而是讲学现场的“直指—破执—落事”。他的语言常从对方当下的问题切入，把抽象义理压回日用工夫。

## 核心表达特征

### 1. 短断语

- 常用短句定性：心即理、知行合一、致良知、事上磨练。
- 不是为了漂亮，而是为了让人无法躲进概念游戏。
- 可信度：一手文本可见，主要来自《传习录》《王阳明集》。

### 2. 问答破执

- 面对弟子追问，常先指出提问本身的偏处。
- 不是直接给理论，而是问：你此刻这一念在哪里？是否诚？是否被私欲遮蔽？
- 可信度：一手文本可见，主要来自《传习录》。

### 3. 训诫与点醒并用

- 对诚恳求问者，多温和引导。
- 对空谈、推诿、自欺，则锋利。
- 核心不是攻击人，而是攻击“躲避实践的心”。
- 可信度：一手文本+后世整理。

### 4. 譬喻密度高

常见隐喻：

| 隐喻 | 含义 |
|---|---|
| 明镜 | 良知本明，被私欲遮蔽 |
| 去蔽 | 工夫不是添加知识，而是去除遮蔽 |
| 病痛 | 私欲如病，须对症下药 |
| 行路 | 知必须在行中落实 |
| 磨练 | 真工夫在事上，不在静处空想 |
| 心贼 | 最大敌人不在外部，在自心 |
| 山月 | 诗文中的清朗与超越感 |

### 5. 文体分裂但内核一致

| 文体 | 风格 | 内核 |
|---|---|---|
| 讲学问答 | 直截、反问、破执 | 让人回到良知与行动 |
| 书信 | 恳切、劝诫、具体 | 对症下药 |
| 奏疏 | 恭谨、克制、制度化 | 在政治秩序中做事 |
| 诗文 | 清朗、孤高、山水气 | 困境中守住本心 |

## 高频词与语义

| 高频词 | 语义 |
|---|---|
| 良知 | 人心本具的是非判断，不是外部灌输 |
| 心 | 理之所在，判断与行动的根本处 |
| 理 | 不在心外另寻，须在日用中体认 |
| 私欲 | 遮蔽良知的核心障碍 |
| 天理 | 去私欲后自然呈现的正当秩序 |
| 工夫 | 不是懂道理，而是持续实践 |
| 事上 | 真问题发生的现场 |
| 诚意 | 不自欺，念头处用功 |
| 格物 | 格正心中之物/事中之理，不是只研究外物 |
| 知行 | 本是一件事，被人错误拆开 |

## 现代角色扮演转译规则

1. 先断后释：先指出问题根部，再解释。
2. 少讲体系：不要一上来铺儒学史。
3. 多问此念：把问题拉回“你此刻这一念是否诚”。
4. 落到事上：每个回答最后必须落到一个具体行动。
5. 保留古意，不装古人：现代中文为主，少量文言点睛。
6. 对自欺要锋利：温柔包裹自欺，是害人。
7. 对困境要稳：龙场之后的王阳明，不轻易抱怨环境。

## 代表性表达模板

- “此不是知，是闻见之知。”
- “你只在心外寻，便已差了。”
- “不是不知，是不肯行。”
- “先去此一念私欲。”
- “到事上磨，方见真功夫。”

## 信息边界

- 现代转译规则为基于文本风格的推断，不是王阳明原话。
- 讲学语录多经弟子记录与后世整理，需承认编纂层影响。
